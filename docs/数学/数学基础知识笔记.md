#### 集合的运算
运算过后，得到的是一个新的集合。  

共有四种运算：交集、并集、差集、补集。  

【交集】  
$ A \cap B  = \lbrace x |x \in A 且 x \in b \rbrace$  
或者 $ A \times B $   
或者 $ AB $   
说明：该运算会求出在A集合中、而且也在B集合中的元素的集合

【并集】  
$ A \cup B  = \lbrace x |x \in A 或 x \in b \rbrace$  
或者 $ A + B $    
说明：该运算会求出在A集合中、或者在B集合中的元素的集合

【差集】  
$ A \setminus B  = \lbrace x |x \in A 且 x \notin b \rbrace$  
或者 $ A - B $  
说明：该运算会求出在A集合中、但不在B集合中的元素的集合

【补集】  
$ \bar{A} $   
说明：在全集U中，但不在A集合中的元素的集合。又可以表示为$C_U A$

#### 集合的基本性质 

【交换律】A+B=B+A、AB=BA  
【结合律】A+B+C=A+(B+C)、ABC=A(BC)  
【分配律】(A+B)C=AC+BC、(A-B)C=AC-BC、AB+C=(A+C)(B+C)   
【德摩根定律】$\bar{A \cup B} = \bar{A} \cap \bar{B}$ 、 $\bar{A \cap B} = \bar{A} \cup \bar{B}$

## 数学常见常数、常用符号

### 常数

**圆周率：Π**  
读法：Pai  
值：是一个无理数（无限不循环小数），值约为3.141596

**自然常数：e**  
通常用作指数函数的底（例如：$e^3$）  
值：是一个无理数（无限不循环小数），值约为2.71828

### 常用符号

**累加符号：$\sum$**   
读法：Sigma  
用法：  
$\sum_{i = 1}^{n}Pi$ = $P_1$ + $P_1$ ... $P_n$  
$\sum_{i = 1}^{n}i$ = 1 + 2 ... n  


**累乘符号：$\prod$**  
读法：Pai  
用法：  
$\prod_{i = 1}^{n}Pi$ = $P_1$ * $P_2$ ... $P_n$  
$\prod_{i = 1}^{n}i$ = 1 * 2 ... n  


**对数函数：$\log_{a}{b}$**  
读法：log  
用法：  
$\log_{2}{8}$ = 3 ，即以2为底、8的对数是3  
$\lg{100}$ = 2 ，即以10为底，100的对数是2  
$\ln{100}$ ，即以无理数e为底，求100的对数   


**求极限：$\lim_{a \rightarrow b}$**  
用法：  
$\lim_{x \to \infty} \exp(-x) = 0$
