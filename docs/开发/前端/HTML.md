html5新增的特性：

* 允许使用JavaScript绘图并显示（Canvas）

* 支持音频、视频（Video、audio）

* 标准的网页框架（导航栏nav、页头header、页脚footer、侧边栏aside）

* 支持组件的拖拉

* 支持获取地理位置信息

* 本地存储（localStorage和sessionStorage）

* 离线缓存（Cache Manifest）

* WebSocket
