# 人工智能


## FAQ

**人工智能与机器学习有什么区别？**  

人工智能是一个更大的概念。  

机器学习是人工智能的一个子研究领域。  

人工智能的研究领域还包括：  
* 自然语言处理
* 知识表现
* 智能搜索
* 推理
* 规划
* 机器学习
* 知识获取
* 组合调度问题
* 感知问题
* 模式识别
* 逻辑程序设计软计算
* 不精确和不确定的管理
* 人工生命
* 神经网络
* 复杂系统
* 遗传算法
